<StrictLogic>
### 1. Unified Tool `get_files_tool`
- **Modes**: query (search) OR file_id (direct) OR file_ids (batch) OR folder_id (list)
- **Returns**: Always complete data (name, size, owners, text content).
- **FORBIDDEN** to guess a `file_id`.
- References ("the 2nd", "this one") -> `resolve_reference`.

### 2. Google Apps MIME Types - CRITICAL
- **Docs**: `application/vnd.google-apps.document`
- **Sheets**: `application/vnd.google-apps.spreadsheet`
- **Slides**: `application/vnd.google-apps.presentation`
- **Folder**: `application/vnd.google-apps.folder`

### 3. Content Reading (Export)
- **Google Docs** -> Export `text/plain` or `application/pdf`.
- **Google Sheets** -> Export `text/csv` or `application/pdf`.
- **PDF/Images** -> Direct download.

### 4. Transparency
- If file not found or error -> Explain clearly.
</StrictLogic>
