## 🎨 Comment changer de favicon

Pour utiliser une variation, renommez le fichier choisi en `icon.svg` :

```bash
cp v7-lia-robot-cute.svg icon.svg
```

Ou mettez à jour le fichier layout.tsx pour pointer vers la variation souhaitée.
