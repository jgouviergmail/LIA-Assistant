## 4. Description of the Service

LIA is a multi-agent conversational AI assistant that provides the following capabilities:

- **Email management**: reading, searching, and drafting emails via Google Gmail, Microsoft Outlook, and Apple iCloud connectors.
- **Calendar management**: viewing, creating, and modifying events via Google Calendar, Microsoft Calendar, and Apple Calendar.
- **Contact management**: searching and viewing contacts via Google Contacts, Microsoft Contacts, and Apple Contacts.
- **Task management**: creating and tracking tasks via Google Tasks and Microsoft To Do.
- **Web search and browsing**: searching for information, visiting web pages, and extracting content.
- **RAG Spaces**: importing and querying personal documents (PDF, text, etc.).
- **Reminders and scheduled actions**: creating reminders and recurring automated actions.
- **Voice control**: voice interaction (speech-to-text / text-to-speech).
- **MCP Protocol**: extending capabilities via external MCP (Model Context Protocol) servers.

The Service uses seven large language model (LLM) providers to generate responses. Responses generated by AI are provided for informational purposes only and do not constitute professional advice (legal, medical, financial, or otherwise) under any circumstances.

**Beta phase**: The Service is currently in beta. As such, it is provided free of charge. The Operator reserves the right to introduce paid plans in the future, with prior notification to existing users. Features may evolve, be modified, or be removed without prior notice during the beta phase.

**HITL (Human-in-the-Loop) mechanism**: For sensitive actions (sending an email, modifying a calendar event, deleting data), the Service requests explicit approval from the User before execution. The User retains full control over actions performed on their behalf.
