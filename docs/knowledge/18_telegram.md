# Telegram Integration

## How do I link my Telegram account to LIA?
You can chat with LIA and receive notifications directly on **Telegram**:

**📲 Setup steps:**
1. Go to **Settings > Messaging Channels**
2. Click **"Link Telegram"**
3. A one-time code (OTP) is generated
4. Open the Telegram bot (a link is provided)
5. Send the OTP command to the bot to link your account

**🔒 Security:**
The OTP is valid for a limited time only. Your Telegram account is securely linked to your LIA account.

## What can I do via Telegram?
Once linked, Telegram becomes a **full communication channel** with LIA:

**💬 Chat:**
• Send text messages to LIA, just like in the web interface
• LIA responds with the same quality and tools (emails, calendar, web search, etc.)
• Voice messages are supported — LIA automatically transcribes them

**🔔 Notifications:**
• Receive proactive notifications (heartbeat, interests) directly on Telegram
• Push notifications work even when the web app is closed
• Same notification settings apply (time windows, frequency, etc.)

## How do I manage my Telegram channel?
In **Settings > Messaging Channels**, you can:

**🎚️ Controls:**
• **Enable/Disable**: Toggle the channel on or off without unlinking
• **Unlink**: Permanently remove the Telegram binding (you can re-link later)

**💡 Tips:**
• You can use both the web interface and Telegram simultaneously
• Conversations are shared — messages sent via Telegram appear in the web chat too
• All your preferences (personality, language, memories) apply across channels
