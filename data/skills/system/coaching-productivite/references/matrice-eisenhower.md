# Matrice d'Eisenhower — Coaching Productivité

## Principe

Classer chaque tâche selon deux axes : **Urgence** et **Importance**.

```
                    URGENT              NON URGENT
              ┌─────────────────┬─────────────────────┐
   IMPORTANT  │   Q1 — FAIRE    │   Q2 — PLANIFIER    │
              │                 │                     │
              │ Crises          │ Projets long terme  │
              │ Échéances       │ Formation           │
              │ Problèmes       │ Stratégie           │
              │ urgents         │ Santé / relations   │
              │                 │                     │
              ├─────────────────┼─────────────────────┤
   NON        │  Q3 — DÉLÉGUER  │  Q4 — ÉLIMINER     │
   IMPORTANT  │                 │                     │
              │ Interruptions   │ Distractions        │
              │ Certains emails │ Réseaux sociaux     │
              │ Certaines       │ Certaines réunions  │
              │ réunions        │ Tâches sans valeur  │
              │                 │                     │
              └─────────────────┴─────────────────────┘
```

## Règles d'application

### Q1 — Faire immédiatement
- **Critère** : Échéance < 24h ET impact significatif si non fait
- **Action** : Traiter maintenant, en premier
- **Objectif** : Minimiser le temps passé ici (anticiper en Q2)

### Q2 — Planifier
- **Critère** : Pas d'échéance immédiate MAIS contribue aux objectifs long terme
- **Action** : Bloquer du temps dans l'agenda, protéger ce créneau
- **Objectif** : **Maximiser le temps passé ici** (c'est la zone de croissance)

### Q3 — Déléguer
- **Critère** : Urgent pour quelqu'un mais ne requiert pas mes compétences
- **Action** : Déléguer ou automatiser, répondre brièvement
- **Objectif** : Réduire le temps passé ici

### Q4 — Éliminer
- **Critère** : Ni urgent ni important
- **Action** : Supprimer, dire non, arrêter de faire
- **Objectif** : Tendre vers zéro

## Questions de tri rapide

Pour chaque tâche, se poser :
1. "Si je ne fais pas ça aujourd'hui, quelles sont les conséquences ?" → Urgence
2. "Est-ce que ça contribue à mes 3 objectifs principaux ?" → Importance
3. "Suis-je la seule personne à pouvoir le faire ?" → Délégation possible ?
