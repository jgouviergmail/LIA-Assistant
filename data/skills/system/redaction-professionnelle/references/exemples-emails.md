# Exemples d'emails — Rédaction Professionnelle

## 1. Demande de rendez-vous

```
Objet : Demande de rendez-vous — [Sujet]

Bonjour [Prénom],

Je me permets de vous contacter afin de solliciter un rendez-vous
concernant [sujet précis].

Seriez-vous disponible la semaine du [date] pour un échange de
[durée estimée] ? Je suis flexible sur les créneaux et m'adapterai
à votre planning.

Dans l'attente de votre retour, je vous souhaite une excellente journée.

Cordialement,
[Signature]
```

## 2. Relance polie

```
Objet : Relance — [Sujet initial]

Bonjour [Prénom],

Je me permets de revenir vers vous concernant mon message du [date]
au sujet de [rappel bref du contexte].

Je comprends que vous puissiez être très sollicité(e), mais je
souhaiterais savoir si vous avez pu prendre connaissance de ma demande.

Restant à votre disposition pour tout complément d'information,
je vous remercie par avance.

Bien cordialement,
[Signature]
```

## 3. Remerciement après réunion

```
Objet : Suite à notre réunion du [date]

Bonjour [Prénom],

Je tenais à vous remercier pour le temps que vous m'avez accordé
aujourd'hui. Notre échange sur [sujet] a été très enrichissant.

Comme convenu, voici un récapitulatif des points abordés :
- [Point 1]
- [Point 2]
- [Point 3]

Les prochaines étapes sont :
- [Action 1] — [Responsable] — [Échéance]
- [Action 2] — [Responsable] — [Échéance]

N'hésitez pas à me faire part de vos remarques ou ajustements.

Bonne fin de journée,
[Signature]
```

## 4. Annonce de mauvaise nouvelle

```
Objet : Retour concernant [Sujet]

Bonjour [Prénom],

Je reviens vers vous suite à votre demande concernant [sujet].

Après analyse approfondie, nous ne sommes malheureusement pas en
mesure de [ce qui est refusé/impossible] en raison de [raison factuelle].

Cependant, nous pouvons vous proposer [alternative concrète] qui
pourrait répondre à votre besoin.

Je reste à votre disposition pour en discuter.

Cordialement,
[Signature]
```
