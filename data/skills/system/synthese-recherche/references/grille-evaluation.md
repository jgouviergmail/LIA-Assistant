# Grille d'évaluation des sources — Synthèse Recherche

## Critères d'évaluation (CRAAP Test adapté)

### 1. Actualité (Currency)
| Score | Critère |
|-------|---------|
| ⭐⭐⭐ | Publié dans les 12 derniers mois |
| ⭐⭐ | Publié dans les 3 dernières années |
| ⭐ | Plus ancien mais toujours pertinent (référence fondatrice) |
| ❌ | Obsolète, données dépassées |

### 2. Pertinence (Relevance)
| Score | Critère |
|-------|---------|
| ⭐⭐⭐ | Traite directement du sujet recherché |
| ⭐⭐ | Couvre un aspect du sujet |
| ⭐ | Tangentiellement lié |
| ❌ | Hors sujet |

### 3. Autorité (Authority)
| Score | Critère |
|-------|---------|
| ⭐⭐⭐ | Expert reconnu, institution de référence, revue à comité de lecture |
| ⭐⭐ | Professionnel du domaine, média spécialisé |
| ⭐ | Source généraliste de bonne réputation |
| ❌ | Auteur non identifié, source douteuse |

### 4. Exactitude (Accuracy)
| Score | Critère |
|-------|---------|
| ⭐⭐⭐ | Données chiffrées, sources citées, méthodologie décrite |
| ⭐⭐ | Affirmations vérifiables, cohérent avec d'autres sources |
| ⭐ | Principalement opinion, peu de preuves |
| ❌ | Contradictions internes, erreurs factuelles |

### 5. Objectivité (Purpose)
| Score | Critère |
|-------|---------|
| ⭐⭐⭐ | Informatif, équilibré, plusieurs points de vue |
| ⭐⭐ | Léger biais identifiable mais factuel |
| ⭐ | Point de vue assumé mais argumenté |
| ❌ | Propagande, publicité déguisée, manipulation |

## Score global

| Total | Fiabilité |
|-------|-----------|
| 13-15 ⭐ | **Haute** — Source de référence |
| 9-12 ⭐ | **Moyenne** — Utilisable avec recoupement |
| 5-8 ⭐ | **Faible** — À corroborer impérativement |
| < 5 ⭐ | **Insuffisante** — Ne pas utiliser |

## Recoupement des sources

Une information est considérée fiable quand :
- **3+ sources indépendantes** la confirment
- Les sources proviennent de **types différents** (académique + média + institution)
- Aucune source ne **contredit directement** sans explication
- Les **données chiffrées convergent** (tolérance ±10%)
